# computer-vision-generative-ai-projects

A comprehensive repository covering Computer Vision and Generative AI projects using Deep Learning, GANs, Diffusion Models, Vision Transformers, and Multimodal AI.


## Features
- Image Classification
- Object Detection
- Image Segmentation
- GAN-Based Image Generation
- Stable Diffusion Projects
- Image-to-Image Translation
- Face AI Applications
- Vision Transformers
- Multimodal AI Systems
- Deployment with FastAPI & Streamlit


## Tech Stack
- Python
- PyTorch
- TensorFlow
- OpenCV
- Hugging Face
- Diffusers
- YOLO
- Detectron2
